How to Play:

  • Click a colored habitat block to select it
  • Click an empty (white) space to move the block there
  • Group same-colored habitats together to create corridors
  • Long corridors (4+ connected blocks) spawn wildlife
  • Some blocks (marked with symbols) cannot be moved
  • Use Auto-Defrag for automatic optimization

Goal: Reduce fragmentation to create continuous wildlife corridors!
